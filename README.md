# competitive-programming-open

競技プログラミングで使用している開発環境・ツール・ライブラリ・提出コードを公開しています。

現在は主に AtCoder 向けのコードやツールを管理していますが、今後は他の競技プログラミングサイト向けの開発環境も追加する予定です。

## 含まれるもの

* 開発ツール・CLI
* テンプレート
* スクリプト
* ライブラリ
* コンテスト終了後のソルバ・提出コード
* その他、競技プログラミングで利用するユーティリティ

## 公開ポリシー

コンテスト期間中のコードやツールは公開しません。

これは、コンテスト中のコード共有が禁止されているためです。公開は原則としてコンテスト終了後に行います。

また、設定ファイルや実験用データなど、一部の内容は非公開または秘匿処理を行う場合があります。可能な限り、コードやツールはそのまま利用できる形で公開します。

## ライセンス

このリポジトリのコードは、学習・研究・参考用途を含め、Apache License 2.0 の範囲で自由に利用できます。

ツールやCLIは、将来的に独立したOSSプロジェクトとして公開することを検討しています。

ライセンス上、連絡は必須ではありませんが、利用・改変・再配布を行う際は、ご連絡いただけると嬉しいです。今後の開発やOSS化の参考になります。
