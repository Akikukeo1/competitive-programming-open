# 問題要約：Conveyor Design (AHC065 - A)

## 1. 概要
$N \times N$ マスの倉庫にランダムに配置された $0$ から $N^2-1$ までの番号付きの箱を、ベルトコンベアの回転操作を用いて、管理番号の小さい順（0, 1, 2, ...）に搬出口からすべて取り出す。

## 2. 制約・環境
- **グリッドサイズ ($N$)**: $20$ 固定
- **搬出口座標**: $(0, N/2) = (0, 10)$
- **設置可能なベルトコンベア数 ($M$)**: $0 \le M \le N^2$
- **ベルトコンベアの長さ ($l_m$)**: $l_m \ge 2$
- **マスの重複制限**: 各マスが独立したベルトコンベアに含まれる回数は**高々2回**まで。
- **最大操作回数 ($T$)**: $0 \le T \le 10^5$ ターン
- **実行時間制限**: 2.0 秒
- **メモリ制限**: 1024 MiB

## 3. 入力形式
入力は標準入力から以下の形式で与えられます。


```

N
a_{0,0}   a_{0,1}   ... a_{0,N-1}
a_{1,0}   a_{1,1}   ... a_{1,N-1}
...
a_{N-1,0} a_{N-1,1} ... a_{N-1,N-1}

```

- $a_{i,j}$ は初期状態でマス $(i,j)$ に置かれている箱の番号（$0 \le a_{i,j} \le N^2-1$）。
- $0$ から $N^2-1$ までの整数がそれぞれちょうど1回ずつ出現するランダムな配置。

## 4. 出力形式
出力は標準出力へ、以下の「ベルトコンベアの設置情報」と「操作列」の2つのパートに分けて出力してください。


```

M
l_0 i_{0,0} j_{0,0} i_{0,1} j_{0,1} ... i_{0,l_0-1} j_{0,l_0-1}
...
l_{M-1} i_{M-1,0} j_{M-1,0} ... i_{M-1,l_{M-1}-1} j_{M-1,l_{M-1}-1}
T
m_0 d_0
m_1 d_1
...
m_{T-1} d_{T-1}

```

### 出力仕様
1. **M (設置するベルトコンベアの個数)**:
   - 続く $M$ 行で、各ベルトコンベアの長さ $l_m$ と、構成するマスの座標を順にスペース区切りで出力する。
   - 隣り合う座標、および末尾と先頭の座標は上下左右に隣接している必要がある。
2. **T (総操作回数)**:
   - 続く $T$ 行で、各ターンに動かすベルトコンベアのインデックス $m_t$（$0 \le m_t < M$）と、回転方向 $d_t \in \{-1, 1\}$ をスペース区切りで出力する。

## 5. ルール・仕様詳細
- **搬出条件**:
  - 各ターンの操作後、搬出口 $(0, 10)$ にある箱の番号が「まだ倉庫に残っている箱の最小値」と一致していれば自動的に搬出される。
  - 初期状態で $(0, 10)$ に箱 `0` がある場合は、最初の操作（0ターン目）の前に自動で搬出される。
- **ループの定義**:
  - $d = 1$ のとき：マス $(i_{m,x}, j_{m,x})$ の中身は $(i_{m,(x+1)\bmod l_m}, j_{m,(x+1)\bmod l_m})$ へ移動する。
  - $d = -1$ のとき：逆方向に移動する。

## 6. スコア計算
取り出せた箱の個数を $B$、操作回数を $T$ とする。

- **すべての箱を搬出できた場合 ($B = N^2$)**:
  $$Score = 10^6 + \text{round}\left(10^6 \log_2 \frac{10^5}{T}\right)$$
  *(操作回数 $T$ が少ないほど高得点)*
- **途中で終了した場合 ($B < N^2$)**:
  $$Score = \text{round}\left(10^6 \times \frac{B}{N^2}\right)$$
